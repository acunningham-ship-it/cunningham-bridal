# Cunningham Bridal — website

Static multi-page site (no build step): `index.html`, `portfolio.html`, `about.html`, `contact.html`, `site.css`, `assets/`.

## Put it on GitHub + free hosting (GitHub Pages)

1. Go to https://github.com/new → name it `cunningham-bridal`, set **Public**, click **Create repository**.
2. On the new repo page, click **uploading an existing file**, drag ALL files in this folder (keep the `assets/` folder), **Commit changes**.
3. Repo **Settings → Pages** → under "Branch" pick `main` + `/ (root)` → **Save**.
4. In ~1 minute the site is live at `https://<your-username>.github.io/cunningham-bridal/`.

To update later: edit files on GitHub or re-upload; Pages redeploys automatically.
